💡 Le bon réflexe

Envoyez le message le jour où le client récupère son chien, tant qu'il est encore ému et content. C'est le moment où le taux d'avis est le plus élevé. Un client satisfait sur trois laisse un avis si on le lui demande — contre presque zéro sans demande.

📇 Destinataires

Tous les emails de vos clients sont récupérés automatiquement (fiches clients + comptes). Les doublons sont retirés. Le message part en copie cachée (Cci) : chaque client reçoit l'email sans voir les autres adresses.

📝 Votre message

« Ouvrir dans ma messagerie » prépare l'email (destinataires en Cci, objet et texte remplis) dans votre logiciel de mail (Gmail, Outlook, Mail…). Vous n'avez plus qu'à cliquer sur Envoyer. Rien n'est envoyé automatiquement.

📢 Publier dans les comptes clients

Publie l'objet + le message ci-dessus comme annonce visible directement dans l'espace client de tous vos clients connectés (en plus ou à la place de l'email). Idéal pour une offre ou une info : ils la voient dès qu'ils ouvrent leur compte.

🖼️ Images du site

Change le hero et les photos de chaque page sans toucher au code. Les nouvelles images sont compressées automatiquement (WebP). « Réinitialiser » remet la photo d'origine, « Masquer » cache l'emplacement.
